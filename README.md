# iPulseExamples

These are simple examples on how to use the User Information feature of <a href="https://ipulseapp.com">iPulse</a>. Both examples generate text that can be easily viewed by hovering over the lower-right corner of iPulse's round window.

The <a href="quote">quote</a> example is a simple quote generator using a JSON endpoint. It generates a plain text file.

The <a href="weather">weather</a> example retrieves weather conditions from the National Weather Service (NWS) using a JSON endpoint. It generates a rich text file.

Both of these folders are designed to work in `/usr/local`. If you put them someplace else, you'll need to update the shell scripts and `cron` paths.

For more information on setting up the User Information feature, see the Help in the iPulse app.
