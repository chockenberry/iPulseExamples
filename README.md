# iPulseExamples

These are simple examples on how to use the User Information feature of <a href="https://ipulseapp.com">iPulse</a>.

The <a href="quote">quote</a> example is a simple quote generator using a JSON endpoint. It generates a plain text file that can be viewed in iPulse.

The <a href="weather">weather</a> example retrieves weather conditions from the National Weather Service (NWS) using a JSON endpoint. It generates a rich text file that can be viewed in iPulse.
